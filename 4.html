MVC web framework

정의

MVC란 Model View Controller의 줄임말로 각 단어의 뜻과 관련된 기능들의 모임이 상호작용하여 작동되는 프레임워크이다.

특징

  1. Model: 데이터베이스에서 저장되는 데이터의 형식을 지정, 저장하고 불러오는 작업들에 관한 코드.
  2. View: 사용자들이 데이터들을 시각적으로 볼 수 있도록하는 코드.
  3. Controller: Model 파트와 View 파트를 연결하여 전반적으로 제어하는 파트.
