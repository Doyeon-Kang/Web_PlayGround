CSS

정의

HTML 등의 마크업 언어로 작성된 문서가 실제로 웹사이트에 표현되는 방법을 정해주는 언어.

CSS의 C가 Cascading의 약자인데, 이는 상위 요소의 스타일 속성을 자손 요소들에게 상속시켜주는 모습이 DOM 트리구조에서 마치 폭포수처럼 내려가는 모습을 닮았기 때문이다.

특징

처음 만들어졌을 때, 웹 브라우저는 HTML이라는 언어만을 해석해서 처리하는 프로그램이었기 때문에 웹 브아루저는 기본적으로 CSS를 HTML로 인식한다. 하지만 CSS는 HTML과 완전히 다른 언어이기 때문에 style 태그를 사용하여 CSS의 사용범위를 지정해줘야 한다. 그리고 style 태그 안에서 어떤 부분에 스타일을 입힐 것인지 지정해주는 코드를 '선택자'라고 한다.

style 태그뿐만 아니라 HTML 소스안에 style 속성을 사용해서 직접 삽입할 수 있다.(여기서 직접 삽입하는 style은 HTML의 속성이다.)

결론적으로 HTML 소스안에 CSS를 작성하는 방법은 두 가지이다. 첫 번째는 style 태그와 선택자를 사용해서 작성하는 것이고, 두 번째는 HTML의 style 속성을 사용하여 직접 작성하는 것이다.

선택자 종류

  • tag 선택자 - 해당 tag name에 속해있는 내용들을 선택. ex). tag name{}
  • id 선택자 - id 태그에 속해있는 내용 중 name과 같은 내용들을 선택.(id 태그는 id 선택자 1개 당 1개의 value만을 가진다.) #name{}
  • class 선택자 - class선택자, class 태그에 속해있는 내용 중 name과 같은 내용들을 선택.(class 태그는 class 선택자 1개가 다수의 value를 가질 수 있다.) .name{}
  • 선택자 우선순위: id > class > tag / 선택범위가 구체적일수록 우선순위가 높다.
  • ,(콤마)선택자 - 콤마 선택자, 어떤 선택자들의 중괄호 내 내용이 같다면 콤마 선택자를 이용해서 코드 양을 줄일 수 있다. ex). h1, a{}

효과 종류

  • color: 폰트 색깔변경
  • text-decoration: 폰트의 효과를 만들거나 없앨 수 있는 속성(none을 붙일경우 효과가 사라진다.)
  • font-size: 폰트의 크기를 조절하는 속성
  • text-align: 텍스트 정렬 조정하는 속성
  • border: 테두리 관련 속성
  • padding: content와 테두리 사이 관련 속성
  • margin: 바깥 여백, 테두리와 테두리 사이 관련 속성
  • display: 화면 전체를 쓰는 태그는 block, 자신의 부피만큼의 크기를 가지는 태그는 inline을 작성한다.
